## 📄 Usage

1. Place your `.txt`, `.md`, or `.pdf` documents in the `documents` folder.
2. Run the script to build the FAISS index:
    ```bash
    python build_index.py
    ```
3. Query the retriever:
    ```bash
    python query_retriever.py "Your search query here"
    ```
